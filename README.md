# PeerConnect: Peer-to-Peer Video Calling (WebRTC)

PeerConnect is a lightweight and secure real-time video calling application built using WebRTC and peer-to-peer (P2P) communication.
This project demonstrates how modern browsers can establish direct media channels without relying on external video servers, offering low-latency, high-quality, and privacy-friendly communication.

## Key Features

- Peer-to-Peer Video & Audio using WebRTC

- No media server required ‚Äî only signaling for handshake

- Secure real-time communication

- Room-based calling with shareable links

- Toggle Audio/Video

- Responsive and minimal UI

- Cross-browser compatibility

## Tech Stack
- Layer	Technology
- Frontend	HTML, CSS, JavaScript
- Real-Time Media	WebRTC (RTCPeerConnection, MediaStream)
- Signaling	Node.js + Express + Socket.io
- Communication	Peer-to-Peer (P2P)
